1. Einführung

Diese Anwendung ermöglicht die Berechnung und Visualisierung der tieffrequenten Schallausbreitung von Lautsprechern im freien Feld.

Hauptaufgabe dieses Programms liegt in der Optimierung der Schallabstrahlung von Lautsprecher Arrays. Somit sind verschiedene visuelle Darstellungen vorhanden, die zur Analyse und Auswertung der Schallabstrahlung dienen.

·  SPL Plot – Farbliche Darstellung des Schalldruckpegels über definierte Fläche

·  SPL Achsen Plots – Schalldruckpegel über eine definierte Achse

·  Polar Plot – Abstrahlung im Fernfeld des Systems, dargestellt im Polardiagramm

·  Impulse Plot – Analyse pro Punkt im Schallfeld: Magnitude, Phase, Impulsantwort

 

Zur Optimierung der Schallabstrahlung dienen verschieden Werkzeuge:

·  Beamsteering mit verschiedenen Algorithmen

·  Windowingfunction mit unterschiedlichen Funktionen

·  Snapshot Engine, um im Optimierungsvorgang die verschiedenen Versionen vergleichen lässt.

 

Import

·  Im Surface Fenster kann mit «import surfaces» exporte aus CAD software geladen werden

·  Im Fenster «manage speaker» können die Lautsprecherfiles verwaltet werden.

·  Unter «Generate» können neue Lautsprecherfiles erstellt werden.  

 

2. Grundlagen

2.1 Programmstart

Nach dem Start des Programms sehen Sie die Hauptansicht mit mehreren Ausgabe Bereichen:

·  SPL Plot: 3D-Darstellung des Schalldruckpegels

·  Y-Axis Plot: Schnittansicht entlang der Y-Achse

·  X-Axis Plot: Schnittansicht entlang der X-Achse

·  Polar Pattern: Polardiagramm der Schallverteilung

 

Für die Eingabe der Daten stehen folgende Ansichtsbereiche zur Verfügung:

·  Surface: Oberflächen für Berechnung erstellen und anpassen

·  Source: Quellen definieren und Positionieren

 

2.2 Menüstruktur

Die Anwendung verfügt über folgende Hauptmenüs:

·  File: Dateioperationen (Neu, Öffnen, Speichern)

·  Setup: Einstellungen und Konfiguration

·  Window: Verwaltung der Fenster und Ansichten

 

3. Dateien verwalten

3.1 Neue Datei erstellen

Erstellen Sie eine neue Projektdatei über File → New oder mit der Tastenkombination Ctrl+Shift+N.

3.2 Datei öffnen

Öffnen Sie eine gespeicherte Projektdatei über File → Open oder mit Ctrl+O.

3.3 Datei speichern

Speichern Sie Ihre Arbeit mit File → Save (Ctrl+S) oder als neue Datei mit File → Save as (Ctrl+Shift+S).

Tipp: Speichern Sie regelmäßig Ihre Arbeit, um Datenverlust zu vermeiden.

 

4. Sources

4.1 Quellen-Fenster öffnen

Öffnen Sie das Quellen-Verwaltungsfenster über Window → Sources oder mit Ctrl+Alt+S.

4.2 Lautsprecher hinzufügen

Im Quellen-Fenster können Sie:

·  Neue Lautsprecher hinzufügen

·  Positionen definieren

·  Lautsprecherdaten importieren

·  Bestehende Quellen bearbeiten

4.3 Lautsprecherdaten verwalten

Verwenden Sie Setup → Manage Speaker (Ctrl+Alt+M), um Lautsprecherdaten zu importieren, zu exportieren oder zu generieren.

4.3 Lautsprecherdaten generieren

Die Daten werden im komprimierten npz‑Format gespeichert.

Über das gleiche Menü können neue Lautsprecherdatensätze erstellt werden. Entweder durch Messdaten in txt Form (Magnitude, Phasenwerte) pro verfügbaren Winkel um den Lautsprecher, oder direkt durch Impulsantwort pro Messpunkt.

Die Daten können mit Frequenzbandfilter oder Fensterung der Impulsantwort geglättet werden. Auch relevant ist der Offset in Zeitebene, wie auch SPL Anpassung, damit korrekte Schalldruckpegel in der Berechnung verwendet werden.

Nach Abschluss wird der Datensatz automatisch im npz‑Format abgelegt und steht für Projekte und Exporte zur Verfügung.

 

5. Surfaces

5.1 Oberflächenfenster

In der Oberflächenübersicht können Flächen oder Gruppen erstellt werden

Flächen die einer Gruppe untergeordnet sind, übernehmen automatisch die Parameter der Gruppe

·  Enable: Mit Enable wird die Fläche für die SPL Berechnung freigegeben.

·  Hide: Hide blendet die Fläche aus.

·  XY: Aktiviert die fläche für die Achsen-SPL Berechnung

Eine Gruppe kann beim Anwählen relativ im Koordinatensystem verschoben werden. Relativer Wert eintragen und mit «Apply Changes» ausführen.

5.2 Oberflächen generieren

·  Öffnen Sie Setup → Manage Surfaces. 

·  Wählen Sie „Add Surface“, um eine neue Fläche anzulegen.

·  Geben Sie Anzahl Punkte und deren Koordinaten ein.   

·  Wenn gewünscht, die Fläche in einer Gruppe ablegen

5.3 Oberflächen importieren

Mit «Import Surface» können Polygone aus txt-Dateien geladen werden. Diese sollten wiefolgt definiert sein:

"Group","Layer"

10.0,10.0,0.0

10.0,-10.0,0.0

-10.0,-10.0,0.0

-10.0,10.0,0.0

 

Mit dem Eintrag «Group» wird das Polygon automatisch in diese Gruppe abgelegt.

Mit «Layer» wird das Polygon beschriftet.

Die Werte sind in X,Y,Z Achsen Reihenfolge.
Sind vier Koordinatenpunkte vorhanden, wird auf Plausibilität geprüft. Wird Test nicht bestanden, wird die Fläche zu zwei 3-Punkte Polygone umgewandelt

Sind mehr als vier Koordinatenpunkte vorhanden, werden daraus 3-Punkte Polygone erstellt.

 

6. Calculation

6.1 Berechnung ausführen

Unter «Settings/Calculation/Update automatically pressure» kann ausgewählt werden, welche der Berechnungs- und Plotmethoden automatisch ausgeführt werden, wenn eine Änderung der Sources oder Surfaces vorgenommen wird.

·  Soundfield: SPL Farb Plot

·  Axis Plot: Plot der SPL Werte in X- und Y-Achse

·  Polar Plot: Fernfeldberechnung und Eintrag in Polardiagram  

·  Impulse Plot: Magnitude, Phase und Impulse Response pro definierten Punkt im Schallfeld

 

Deaktivierte Methoden werden bei Parameteränderungen nicht automatisch berechnet und geplottet.

Manuelle Berechnung kann mit dem Calculate-Button oder der Taste C ausgeführt werden.

6.2 Berechnungsmodi

Es stehen verschiedene Berechnungen und Ausgaben zur Verfügung

 

6.2.1 SPL Plot / Phase Plot

Im SPL Plot wird der Schalldruckpegel pro definierten Punkt im Schallfeld berechnet (Superposition) und als Farbe ausgegeben.

Nur Surfaces, die «enabled» und «nicht hide» sind, werden berechnet und geplottet.

 

·  Frequenz: Der Berechnete Frequenzbereich kann unter «Settings/….» gewählt werden

·  Auflösung der Berechnung: In unter «Settings/….» kann die Anzahl Berechnungspunkte eingestellt werden, die für die Frequenz und Phasenmittelung verwendet werden.

 

Die Farben werden in der Colorbar links im Fenster den zugehörigen Schalldruckpegeln zugewiesen.

·  Color Range: In «Settings/Mapping/SPL color range» kann die Farbskala manuell eingestellt werden.

·  SPL Mode: In «Settings/Mapping/SPL color range» kann mit die Farbgebung mit «Gradient» oder «Colorstep» umgestellt werden.

·  Automatische Farbanpassung: Mit Doppelclick auf die Colorbar wird der höchste SPL Wert im Schallfeld detektiert und Skala angepasst.

 

Im Phase Plot wird die relative Phase (Grad) der einzelnen Systeme zueinander in Farbe dargestellt.

 

6.2.2 Axis Plots

Im Axis Plot wird der Schalldruckpegel pro definierter Punkt auf Achse im Schallfeld berechnet (Superposition) und in 2D Grafik dargestellt.

Nur Surfaces, die «enabled» und «nicht hide», sowie «XY» aktiv sind, werden berechnet und geplottet.

 

·  Achsposition: In «Settings/…» kann die Position der Achsen manuell eingestellt werden.

 

6.2.3 Polar Plot

Im Polar Plot wird das Abstrahlverhalten aller aktiven Systeme die «enabled» und «nicht Hide» sind berechnet.

Die Berechnung erfolgt unter Fernfeldbedingungen und ergibt den Schalldruckpegel auf einem Kreis um das System.

 

·  Frequenz pro Farbe: In «Settings/…» kann die Frequenz der Farben im Plot eingestell werden.

 

 

6.2.4 Impulse Plot

Im Impulse Plot Fenster können Punkte im Schallfeld definiert werden.

Aus der superpositionierten Magnitude und Phase wird die Summenimpulsantwort berechnet.

Zusätzlich zu den Summen Mag- / Phase- / IR-Daten werden auch die Daten der einzelnen Systeme geplottet.

Neben der Impulsantwort gibt es das Fenster «Arrival Times». Pro Source wird der Pegel und die Laufzeit zum Impulspunkt berechnet und auf einer Zeitleiste dargestellt.

8. Visualisierung

8.1 3D-Plot Interaktion

Im 3D-SPL-Plot können Sie:

·  Mit der Maus rotieren (linke Maustaste)

·  Zoomen (Mausrad)

·  Verschieben (rechte Maustaste oder Shift + linke Maustaste)

8.2 Ansichten fokussieren

Verwenden Sie die folgenden Tastenkombinationen, um bestimmte Ansichten zu fokussieren:

·  Ctrl+1: Focus SPL

·  Ctrl+2: Focus Yaxis

·  Ctrl+3: Focus Xaxis

·  Ctrl+4: Focus Polar

·  Ctrl+5: Default View

8.3 Overlays und Einstellungen

Im 3D-Plot können verschiedene Overlays aktiviert werden:

·  Achsen-Overlays

·  Impuls-Overlays

·  Oberflächen-Overlays

Hinweis: Die Transparenz der Achsen-Overlays kann in den Einstellungen angepasst werden.

10. Tipps & Tricks

10.1 Optimale Arbeitsweise

·  Definieren Sie zuerst Ihre Quellen und deren Positionen

·  Laden Sie die entsprechenden Lautsprecherdaten

·  Konfigurieren Sie die Berechnungsparameter

·  Führen Sie Testberechnungen durch, bevor Sie komplexe Szenarien berechnen

10.2 Performance-Optimierung

Für große Berechnungen können Sie die Berechnungsauflösung in den Einstellungen anpassen, um die Berechnungszeit zu verkürzen.

10.3 Visualisierung

Experimentieren Sie mit verschiedenen Overlays und Ansichten, um die besten Einblicke in Ihre Daten zu erhalten.

Wichtig: Bei sehr großen Datenmengen kann die Visualisierung Zeit in Anspruch nehmen. Haben Sie Geduld oder reduzieren Sie die Auflösung.
