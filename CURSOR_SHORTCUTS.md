# Cursor Keyboard Shortcuts

## Code-Navigation

### Datei-Navigation
- **⌘ + P** - Schnelles Öffnen von Dateien (Quick Open)
- **⌘ + ⇧ + P** - Befehlspalette öffnen
- **⌘ + G** - Zu einer bestimmten Zeile springen
- **⌘ + ⇧ + O** - Zu Symbol in aktueller Datei springen (Funktionen, Klassen, etc.)
- **⌘ + T** - Zu Symbol im gesamten Projekt springen

### Suche
- **⌘ + F** - In aktueller Datei suchen
- **⌘ + ⇧ + F** - Projektweit suchen
- **⌘ + ⇧ + H** - In aktueller Datei ersetzen
- **⌘ + Option + F** - Projektweit ersetzen

### Navigation im Code
- **F12** - Zur Definition springen
- **⌘ + F12** - Zur Implementierung springen
- **⌘ + K, F12** - Definition in Seitenleiste öffnen
- **⇧ + F12** - Alle Referenzen anzeigen
- **⌘ + ⇧ + .** - Nächste Referenz
- **⌘ + ⇧ + ,** - Vorherige Referenz
- **⌘ + -** - Zurück navigieren
- **⌘ + ⇧ + -** - Vorwärts navigieren
- **⌘ + ↑** - Cursor nach oben scrollen (ohne zu bewegen)
- **⌘ + ↓** - Cursor nach unten scrollen (ohne zu bewegen)

### Mehrfach-Cursor
- **⌘ + D** - Nächstes Vorkommen auswählen
- **⌘ + ⇧ + L** - Alle Vorkommen auswählen
- **⌘ + Option + ↑/↓** - Cursor oberhalb/unterhalb einfügen
- **⌘ + U** - Letzte Cursor-Aktion rückgängig machen

## Code ein- und ausklappen (Code Folding)

### Einzelne Bereiche
- **⌘ + Option + [** - Aktuellen Bereich einklappen
- **⌘ + Option + ]** - Aktuellen Bereich ausklappen
- **⌘ + K, ⌘ + 0** - Alle Bereiche einklappen
- **⌘ + K, ⌘ + J** - Alle Bereiche ausklappen

### Spezifische Ebenen
- **⌘ + K, ⌘ + 1** - Alle Bereiche der Ebene 1 einklappen
- **⌘ + K, ⌘ + 2** - Alle Bereiche der Ebene 2 einklappen
- **⌘ + K, ⌘ + 3** - Alle Bereiche der Ebene 3 einklappen
- (usw. bis Ebene 7)

### Erweiterte Folding-Optionen
- **⌘ + K, ⌘ + /** - Alle Kommentarblöcke einklappen
- **⌘ + K, ⌘ + 8** - Alle Bereiche außerhalb der aktuellen Auswahl einklappen
- **⌘ + K, ⌘ + 9** - Alle Bereiche außerhalb der aktuellen Auswahl ausklappen

## Editor-Navigation

### Zeilen-Navigation
- **⌘ + →** - Zum Zeilenende
- **⌘ + ←** - Zum Zeilenanfang
- **⌘ + ↑** - Zum Dateianfang
- **⌘ + ↓** - Zum Dateiende
- **Option + →** - Ein Wort nach rechts
- **Option + ←** - Ein Wort nach links

### Auswahl erweitern
- **⌘ + ⇧ + →** - Bis Zeilenende auswählen
- **⌘ + ⇧ + ←** - Bis Zeilenanfang auswählen
- **⌘ + ⇧ + ↑** - Zeile nach oben auswählen
- **⌘ + ⇧ + ↓** - Zeile nach unten auswählen
- **Option + ⇧ + →** - Wort nach rechts auswählen
- **Option + ⇧ + ←** - Wort nach links auswählen

## Tabs und Editor-Gruppen

- **⌘ + W** - Aktuellen Tab schließen
- **⌘ + K, W** - Alle Tabs schließen
- **⌘ + K, ⇧ + W** - Alle Tabs außer dem aktuellen schließen
- **⌘ + ⇧ + T** - Zuletzt geschlossenen Tab wieder öffnen
- **⌘ + 1/2/3** - Zwischen Editor-Gruppen wechseln

## Nützliche Tipps

- **⌘ + K, ⌘ + S** - Tastenkombinationen anzeigen/bearbeiten
- **⌘ + ,** - Einstellungen öffnen
- **⌘ + B** - Seitenleiste ein-/ausblenden
- **⌘ + J** - Terminal ein-/ausblenden
- **⌘ + `** - Zwischen Terminal und Editor wechseln

## Hinweise

- Auf macOS werden **⌘** (Command), **⌥** (Option/Alt), **⇧** (Shift) und **⌃** (Control) verwendet
- Einige Shortcuts können je nach installierten Erweiterungen variieren
- Alle Shortcuts können in den Einstellungen angepasst werden: **⌘ + K, ⌘ + S**
